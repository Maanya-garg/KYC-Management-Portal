# KYC Management Portal

A full-stack KYC Management Portal developed during my Software Engineering Internship at IFFCO TOKIO General Insurance Co. Ltd.

## Features
- Secure Login
- Dashboard
- Create KYC
- Search KYC
- Update KYC

## Tech Stack

### Frontend
- Angular
- TypeScript
- Angular Material
- HTML
- CSS

### Backend
- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- REST APIs

### Database
- H2 Database

## Tools
- Swagger (OpenAPI)
- Maven
- Git
- GitHub
- Postman

## Getting Started

### Backend
```bash
mvn spring-boot:run
```

### Frontend
```bash
cd kyc-frontend
npm install
ng serve
```

Visit:
```
http://localhost:4200
```
